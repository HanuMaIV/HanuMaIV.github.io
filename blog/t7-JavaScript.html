Ruby Hashes vs. JavaScript Objects

Syntax

Let's take a look at Ruby hashes and Javascript Objects. The Ruby hash and the Javascript Object look very similar but are quite different once you take a closer look. You will see what I mean as I create both below.

Creating a Ruby Hash and Javascript Object

#Ruby Hash
  brendan = {name: "Brendan", height: "6'1", weight: 170}


//Javascript Object
var brendan = {name: "Brendan", height: "6'1", weight: 170 }

See how similar they look? Let's look at each in closer detail.

The Ruby Hash

A Ruby hash is simply a collection of key value pairs. Much like the simple array. The difference being that instead of a number index a hash has a "dictionary" type index in which a word is the "key" that maps to a certain "value". For example:

#Simple array
brendan = ["Brendan", "6'1", 170]

#Simple Array values are accessed with an integer index
array[0]
=> "Brendan"

array[1]
=> "6'1"

array[2]
=> 170

##############################
#Ruby Hash
brendan = {name: "Brendan", height: "6'1", weight: 170}

#Values are accessed using a dictionary type index
brendan[:name]
=> "Brendan"

brendan[:height]
=> "6'1"

brendan[:weight]
=> 170

It's apparent that the Ruby simple array and hash are quite similar. Both are just key and value pairs with a different way of indexing values.

Javascript Object

Javascript objects are standalone entities with properties and type. The properties of javascript objects are similar to variables but in this case the variables are associated to the parent object. Properties of Javascript objects can be accessed by using dot notation. Let's see this in action:

var brendan = {name: "Brendan", height: "6'1", weight: 170 }

brendan.name
=> 'Brendan'

brendan.height
=> '6\'1'

brendan.weight
=> 170

When functions are associated with objects they are called methods. Lets take a look:

var brendan = {name: "Brendan",
               height: "6'1",
               weight: 170,

               // Defining method here
               introduce: function() {
               console.log("Hello my name is " + this.name + ".");
             }
};

// Calling Method
brendan.introduce();
=> Hello my name is Brendan.

Summary

You can start to see that Javascript objects are much more complicated than just a simple "key" / "value" pairing (as in the Ruby Hash). And we have explored some of the differences between the two. There is much more to learn about both Ruby Hashes and Javascript objects.
