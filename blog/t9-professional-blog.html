Professional Blog

This week I was tasked with using feedback to make my blog more professional. After initially sending out emails and tweets, I was able to recognize that the colors needed to be toned down.

I originally had a different color for each page. I can see how that was a little too much. I settled on a body color of "#95a5a6" which I feel is a good balance of playfulness and easy on the eyes (before I had some pretty alarming colors). One of my friends (Karli) was able to help me out with reducing contrast to find that color. It was nice to have that help.

Originally I had also included funny faces on every page of my site on my profile picture. I toned that down also. The playful picture now only appears on my contact page.

I also took time this week to upgrade my site to use cname, A records and MX (mail exchanger)records:

  • brendanmiranda.com A Record 192.30.252.154
  • brendanmiranda.com A Record 192.30.252.153 (backup)
  • www.brendanmiranda.com --> CNAME brendanmiranda.com (this is to get rid of the www, which is a personal preference of mine).
  • mail.brendanmiranda.com A record 198.20.70.218 (friendly name for accessing my mail server through mail clients)
  • MX Record for brendanmiranda.com --> 198.20.70.218 (this is so my email still works)
  • Created a file on the root of my github repo named "CNAME", brendanmiranda.com is listed on line 1. This tells github to respond to queries to brendanmiranda.com

The dns was pretty simple the only thing was that I forgot that my previous MX record pointed to brendanmiranda.com. That was causing errors because it was pointing to 192.30.252.154 (my github repo) because that is what brendanmiranda.com is resolving to now. I had to update the MX record to point to 198.20.70.218 in order to get my mail working again. Everything is good now.
