Class Methods

Class methods are global methods. You can call them without the need to create an object instance.

We are all pretty used to instance methods. They are usually one of the first concepts we learn even if we don't specifically call them instance methods.

class User
  def instance_method
    p "I am an instance method"
  end
end

user = User.new
user.instance_method

=> "I am an instance method"

Here we created a class called User with just one simple instance method that displays text to the output. Notice how we have to instantiate the class into an object in order for us to run the instance method? What would happen if we tried to call the method without instantiating? Let's give it a try.

class User
  def instance_method
    p "I am an instance method"
  end
end

User.instance_method

=> class_method.rb:31:in `instance_method': wrong number of arguments

As you can see we get a nice little error. Why did this happen? It's because the instance method only gets created when an object is instantiated. Then the method is available to the object but not the class. You can actually see the scope of the method by using the self method.

class User
  def instance_method
    p self
  end
end

user = User.new
user.instance_method

=> #User:0x007fd721074950

You can see that "self" in the context of the instance method refers to the actual object created by the class but not the class itself.

Let's create a class method now by simply adding "self." before the method name.

class User
  def self.class_method
    p "I am a class method, hear me roar!"
  end
end

User.class_method

=> "I am a class method, hear me roar!"

Notice how we can now call the class method without having to instantiate the class using the User.new statement?

Now let's look at "self" in the context of the class method

class User
  def self.class_method
    p self
  end
end

User.class_method

=> User

Looks like self exists in the context of the actual User object and not an object instantiated with the user class.

Also keep in mind that class methods are not available to objects instantiated with that class. For example:

class User
  def self.class_method
    p self
  end
end

user = User.new
user.class_method

=> undefined method `class_method' for #<User:0x007fcf1c8748d8> (NoMethodError)

You can clearly see that the class method is not available to the object instantiated with the User class.
