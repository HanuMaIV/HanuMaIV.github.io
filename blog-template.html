What is version control?

By Brendan Miranda

2.10.15

Version control is software that is installed on a computer that allows a software developer to monitor changes to the files and folders on a computer. Have you ever used the undo button in a software program? If the answer is yes, then you are on your way to understanding version control. Version control is much like that undo button, but injected with steroids.
